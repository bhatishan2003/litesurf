Installation
============

- **From pypi**:
  .. code-block:: bash

    pip install litesurf

- **Clone the repository**:

  .. code-block:: bash

      git clone https://github.com/bhatishan2003/litesurf.git
      cd litesurf

- **Install the package**:

  .. code-block:: bash

      pip install .

- **For development (editable mode)**:

  .. code-block:: bash

      pip install -e .
