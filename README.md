## Installation

-   Clone the repository:

    ```bash
    git clone https://github.com/bhatishan2003/minibrowser.git
    cd minibrowser
    ```

-   Install the package:

    ```bash
    pip install .
    ```

-   For development (editable mode):

    ```bash
    pip install -e .
    ```

## Usage

### Basic Python Usage

```bash
python3 minibrowser.py
```

### Command Line Usage

```bash
minibrowser
minibrowser --file test.html

```
